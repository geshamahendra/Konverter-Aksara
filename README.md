Konversi dari Latin ke Aksara Jawa-Bali-Kawi. Akan ditambahkan aksara lain jika memungkinkan

Cara penggunaan:

Untuk mode default pengetikan menggunakan latin IAST yang ditulis dalam file input.txt lalu mengeksekusi file main.py, hasil output akan muncul didalam folder output untuk aksara jawa-bali-kawi dan ,atin yang telah dinormalisasi.

Catatan:
1. sampai saat ini terdapat 2 model utama yang didukung secara resmi yaitu kakawin & normal, pemilihan mode dilakukan dengan menambahkan simbol +<mode>+ pada baris pertama file input, contoh:


===========input.txt===========
+kakawin+

{rajanī}:25.108
⏑ ⏑ ⏑ | ⏑ — ⏑ | — ⏑ ⏑ | ⏑ — ⏑ | ⏑ — ⏑ | ⏑ ⏓

kuya ṅ amariṅ-mariṅ mararakan mara riṅ maḍaké,

===============================

2. Penulisan latin menggunkanan konsonan lowercase, adapun vokal sangat sensitive case karena menentukan swara atau sandhangan.
